Node.JS + NPM

Additional modules can be accessed through NPM. Yes, you will have to hone your command line skills! The only three modules installed for this app are:

  • EJS
  • Express
  • MySQL

Initializr + Bootstrap

The front-end is built on the Twitter Bootstrap version of Initializr (formerly HTML5 Boilerplate). It's fully-responsive and bad-ass! To get started, do the following:

  • Create an OpenShift account
  • Create a Node.JS app and fork directly from GitHub
  • Add the MySQL and phpMyAdmin cartridges
  • Clone the OpenShift repo to your computer
  • Upload surfers.sql into phpMyAdmin
